FAQ

How to download your projects?

To install our projects, you need to go to the page https://GoodModsLab-Official.github.io/services.html. Then follow the instructions. If you have any questions about installing the application, look here

How to download our apps.

  1. Download the installation file to your device.  You must install the project file on your device through the market where our official application is located.
  2. Allow the installation of the file. It is possible that your device will be against installing the file, you must allow it through the device settings (For example, on an android device, this is the <Applications> item).
  3. Find where the application file was installed. You must find the file and complete the installation, after which you can use our applications. (On Android devices, there may be a further blocking of the application, for this you need to allow installation from third-party sources).
