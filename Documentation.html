FAQ

Как установить наши проекты?

Чтобы установить один из наших проектов посетите страницу: https://GoodModsLab-Official.github.io/services.html. Затем следуйте инструкциям. Если у вас есть вопросы по установке приложения, смотрите здесь.

Как установить приложение

  1. Загрузите установочный файл на свое устройство.  Вы должны установить файл проекта на свое устройство через маркет, где находится наше официальное приложение.
  2. Разрешите установку файла. Возможно, что ваше устройство будет против установки файла, необходимо разрешить это через настройки устройства (Например, на устройстве Android это пункт «Приложения»).
  3. Найдите, где был установлен файл приложения. Вы должны найти файл и выполнить установку, после чего вы сможете пользоваться нашими приложением. (На устройствах Android возможна дальнейшая блокировка приложения, для этого нужно разрешить установку из сторонних источников).
